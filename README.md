## Server Status & Monitoring

Tools for checking server status, monitoring uptime, and tracking player activity.

- [Minecraft ServerHub](https://minecraft-serverhub.com) - Free server monitoring platform with real-time status checks, player tracking, and developer API. Supports Java and Bedrock servers. ([API Docs](https://minecraft-serverhub.com/developers)) ([Statistics](https://minecraft-serverhub.com/statistics))
- [MCStatus.io](https://mcstatus.io) - Simple server status checker with API.
- [mcsrvstat.us](https://mcsrvstat.us) - Lightweight Minecraft server status checker.
- [Minecraft Server Status](https://mcsrv.us) - Basic server ping tool.

## Developer APIs

APIs for building Minecraft-related applications, bots, and integrations.

- [Minecraft ServerHub API](https://minecraft-serverhub.com/developers) - Free REST API for querying any Java/Bedrock server status. No API key required. Returns player count, MOTD, version, latency, and server icon.
- [Mojang API](https://wiki.vg/Mojang_API) - Official Minecraft account and profile API. UUID lookups, skin data, and authentication.
- [Crafatar](https://crafatar.com) - Player avatar and skin render API. Fast CDN-cached responses.
- [CraftHead](https://crafthead.net) - Ultra-fast player head renders via Cloudflare Workers.
- [MCStatus.io API](https://mcstatus.io) - Alternative server status API.

## Web Tools

Browser-based tools for server owners and players.

- [Minecraft ServerHub Badge Generator](https://minecraft-serverhub.com/tools/server-badge) - Create embeddable server status badges in rounded, flat, or minecraft styles. Outputs HTML, Markdown, and BBCode.
- [Minecraft ServerHub MOTD Creator](https://minecraft-serverhub.com/tools/motd-creator) - Visual MOTD editor with live preview. Supports all Minecraft color codes and formatting.
- [Minecraft ServerHub Server Explorer](https://minecraft-serverhub.com/explore) - Browse and search 5000+ active servers by gamemode, version, and region.
- [MineServ.top](https://mineserv.top) - Server listing directory.
- [Minecraft Server List](https://minecraft-server-list.com) - Community server directory.

## npm Packages

Node.js packages for Minecraft server integrations.

- [minecraft-server-badge](https://www.npmjs.com/package/minecraft-server-badge) - Generate embeddable Minecraft server status badges. Returns badge URLs, HTML, Markdown, and BBCode. ([Source](https://github.com/DraginoxXx/minecraft-server-badge))
- [minecraft-server-util](https://www.npmjs.com/package/minecraft-server-util) - Query Minecraft servers using the SLP protocol.
- [node-minecraft-protocol](https://www.npmjs.com/package/minecraft-protocol) - Full Minecraft protocol implementation for Node.js.

## Discord Bots
